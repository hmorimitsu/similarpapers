Privacy policy

This site uses cookies (read about what is a cookie) to monitor the website traffic and keep some data for improving the user experience. These cookies are not used to store personal information. More specifically, this website uses the following cookies:

  • Google Analytics: for monitoring the website traffic. These cookies are anonymized (read about Analytics IP anonymization).
  • Cloudflare: for improving the performance of the website. These cookies are also anonymized (read about Cloudflare cookies).
  • Conference choices: for improving the usability. These cookies just store information about which conferences the user selected to filter the similar results.

If you want, you can disable the cookies for this website and it should still work normally, except that it will not remember your choices.

The way to disable the cookies depends on the browser you are using, and you should check its respective documentation to find the instructions.

Conferences

Conferences on CV, ML, and AI that provided open access from the last 5 years were selected based on the ranking from Google Scholar. Please do not ask to include more conferences to the list. The choices are restricted to keep the use of resources to a minimum.

If you want to use other conferences, you can fork the source code and run a server with your selection.
